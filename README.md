# 3ds-Rainbow-Light-PERMENANT-WITH-MCU-BRICKER-
Use this repo to get a permenent rainbow light on your 3ds notification light after boot!

1: download the .3dsx file in the repository

2: power on your 3ds, go to universal-updater, and search for MCU Bricker

3: download the first file, then power off your 3ds

4: rename boot.3dsx in the root folder of your system's SD card to luma.3dsx and put the boot.3dsx file we downloaded in step one into the root folder

5: hold START while powering on to open the luma3ds menu

6: click any option that says something along the lines of, "launch luma3ds at boot" (will get updated with the correct term once i have the recources) 

7: Go down to save and exit and click it!
